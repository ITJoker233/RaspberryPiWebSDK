# 🍕RaspberryPiWebSDK
> 树莓派 WebSDK，简易，快速 python3.6+

## 📌用法

### 🛠配置环境
```
pip3 install -r requirements.txt
```

### 🔧配置文件
修改config.json文件中的ApiKey 为你自己的请求key
```
{
    "Version": 100,
    "ApiKey": "default@Raspberry"
}
```

### 🚀运行RaspberryPiWebSDK
```
sudo python3 run.py
```
